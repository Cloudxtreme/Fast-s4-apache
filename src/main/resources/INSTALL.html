Installation instructions

To install the real-time tracking code:

  1. Copy and paste the following right before the closing '</body>' tag of all the pages you want to track.
  2. Replace your-domain-here by the domain of your website, e.g. aljazeera.com

Optionally, you can set the owa_sampleRate to any value in ]0,1] to sample a fraction of visits.
